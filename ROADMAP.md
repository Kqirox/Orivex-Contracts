# Orivex Contracts Roadmap

This document outlines the planned development milestones, quarterly goals, and key feature deliverables for the Orivex Contracts project.

## Overview

Orivex is a learning-and-rewards protocol built on Soroban (Stellar) that enables soulbound badges, USDC rewards, quest management, and decentralized governance. The roadmap is structured around three core milestones: v0.2, v0.3, and v1.0.

## v0.2 - Q3 2026 (Target: September 30, 2026)

### Core Objectives

- Complete core feature set for all modules
- Establish comprehensive testing coverage
- Implement basic security measures

### Status Table

| Category              | Status      | Target Date        |
| --------------------- | ----------- | ------------------ |
| Core Contracts        | In Progress | July 31, 2026      |
| Comprehensive Testing | Planned     | August 31, 2026    |
| Security Audit Prep   | Planned     | September 15, 2026 |
| Documentation         | In Progress | September 30, 2026 |

### Key Feature Deliverables

- **Core Features**
  - Finalize `badge-nft` contract with soulbound badge issuance and revocation
  - Complete `course-registry` with full course lifecycle management
  - Finalize `reward-pool` with USDC funding and distribution
  - Complete `stake-vault` with token staking and lockup mechanisms
  - Finalize `governance` with badge-weighted voting and proposal lifecycle
  - Complete `quest-engine` with build and explore quest management
- **Testing**
  - Achieve 90%+ test coverage across all contracts
  - Implement integration tests for cross-contract interactions
  - Add property-based testing for critical functions
  - Set up continuous integration (CI) pipeline for automated testing
- **Security**
  - Conduct internal security audit of all contracts
  - Address all critical and high-severity issues
  - Implement formal verification for critical functions
  - Prepare for external security audit
- **Documentation**
  - Complete inline code documentation
  - Create developer guides for each contract
  - Establish API documentation
  - Write deployment guides
