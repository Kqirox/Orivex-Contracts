# Glossary

| Term | Definition |
|---|---|
| Soulbound | A token that cannot be transferred between addresses. |
| Authorizer | The protocol admin address allowed to perform privileged operations. |
| Spender | A contract that the RewardPool has whitelisted for payouts. |
| Multiplier | The basis-points scale applied to a learner's quest payout, fetched from StakeVault. |
| Quest-counter | The monotonically-increasing ID assigned to each new QuestEngine quest. |
